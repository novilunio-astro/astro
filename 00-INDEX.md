# База знаний: Астрология для стартапа

> Собрано: июль 2026. Все ресурсы — легальные и бесплатные (или с бесплатным уровнем).

## Структура базы

| Файл | Содержание |
|------|-----------|
| `01-osnovy-astrologii.md` | Основы западной астрологии: планеты, знаки, дома, аспекты |
| `02-natalnaya-karta.md` | Натальная карта: построение и чтение, порядок разбора |
| `03-tranzity-i-prognozirovanie.md` | Транзиты, прогрессии, солнечные возвращения — методы прогноза |
| `04-matrica-sudby.md` | Матрица судьбы: метод, расчёт, арканы |
| `05-kursy-i-knigi.md` | Каталог бесплатных курсов и книг (RU + EN) |
| `06-instrumenty-i-api.md` | Сервисы расчёта, библиотеки и API для продукта |

## Как пользоваться

- Файлы 01–04 — конспекты-справочники по темам: терминология, структура, ключевые принципы. Это каркас, который можно наращивать материалами из курсов.
- Файл 05 — куда идти учиться: курсы и книги с пометками по уровню и языку.
- Файл 06 — техническая часть для стартапа: чем считать карты и транзиты программно.

## Как дополнять базу

1. Новые заметки по теме — добавляй в соответствующий тематический файл под нужный заголовок.
2. Новые найденные курсы/книги — в таблицы файла 05.
3. Конспекты пройденных курсов — можно заводить отдельными файлами вида `notes-<название>.md` и ссылаться на них отсюда.
